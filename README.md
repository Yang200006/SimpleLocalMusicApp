# 簡易本地音樂撥放器

<h3>功能介紹</h3>
<P>● 點擊主畫面上的歌曲後，播放被點擊的歌曲。</P>
<P>● 點擊主畫面下方的三個按鈕，可切換、暫停、播放歌曲。</P>
<P>● 點擊主畫面右上角的尋按鈕後，輸入關鍵字可搜尋相關的歌曲。</P>

<h3>歌曲來源</h3>
儲存在Android Studio模擬器的sdcard下的Download資料夾
